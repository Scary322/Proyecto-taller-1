# **Cuida a tu compañero**

## HTML, CSS, SCSS, GIT, GITHUB, JAVASCRIPT.

### En la actualidad el 56% de la población mundial tiene una mascota, y todas ellas necesitan tener a la mano información precisa a cerca de donde llevar a sus mascotas dependiendo de sus necesidades, desde un veterinario, a un lugar de cuidado, y esta es una base para crear una página que supla esas necesidades.

### Para poder usar esta plantilla debes clonar mi repositorio, abrir visual studio code e instalar la extensión **live server**, después inicia la extensión (Debe aparecer en la parte inferior derecha de visual studio code) y listo, tendrás una vista previa del proyecto.
